VAST Data AI Assessment

Lead generation landing page

UX , UI, User testing, Accessibility, Handoff

The AI readiness assessment, created for VAST Data, is a valuable tool that helps potential clients evaluate their current AI capabilities. By identifying gaps or shortcomings in their existing AI setup, this assessment provides organizations with a clearer understanding of their AI readiness. Users can navigate through a series of questions, resulting in an engaging and interactive experience.

After completing the assessment, users receive a comprehensive report that highlights both strengths and areas needing improvement. With a clear presentation of results, the AI readiness assessment serves as an essential resource for organizations looking to optimize their AI infrastructure and make informed decisions while connecting to VAST Data's solutions.

UX Exploration

Since the client already had a clear vision of the tool they needed for their marketing lead generation strategy, we were able to quickly start creating high-fidelity wireframes. They also provided assets from their design library, which facilitated this process. As a result, we could rapidly iterate and collaborate to define the final scope of the MVP.

UI Production

Using VAST's existing design system as a foundation, we began to develop the final user interface. We collaborated closely with the in-house design team to validate new components and refine existing ones. Additionally, we defined transitions and micro-animations to ensure a smooth and cohesive user experience.

Micro interactions and transitions

In this project, we focus on micro-interactions and transitions to improve user experience by providing clear feedback and aiding navigation. Subtle animations help make the interface more intuitive and consistent, which can enhance usability. By incorporating these design elements, we aim to create a smoother interaction for users as they navigate through the tool.

Responsive Design and PDF dynamic generation

In addition to implementing responsive breakpoints, we developed a dedicated CSS for print to enhance the PDF generation process. This specialized stylesheet includes specific instructions that adapt the layout for paper format, ensuring the content is optimized for printing and appears polished and professional. Additionally, the print CSS addresses animated elements and colored components, transforming them into a more print-friendly format by simplifying animations and adjusting colors for clarity and readability on paper.
